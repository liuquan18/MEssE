# MEssE v1.0 - Monitoring Interface

This folder contains the web-based monitoring interface for MEssE v1.0, which provides real-time visualization of ICON simulation progress and neural network training.

## Quick Start

```bash
cd /work/mh0033/m300883/Project_week/MEssE/scripts/plugin/monitor
./start_monitor.sh
```

Then access: **http://localhost:5000**

## Files in This Folder

- **monitor_server.py** - Flask web server
- **start_monitor.sh** - Launcher script (executable)
- **templates/monitor.html** - Web interface UI
- **MONITOR_README.md** - Complete documentation
- **QUICK_START.md** - Quick reference guide

## Features

✨ Real-time ICON simulation monitoring
✨ Live training loss visualization with charts
✨ Auto-refresh every 2 seconds
✨ Modern, responsive UI

For detailed documentation, see [MONITOR_README.md](MONITOR_README.md) or [QUICK_START.md](QUICK_START.md).
